# Сценарий инсталляции на виртуальную машину с ubuntu 16.04

## Базовые зависимости
* python3 
* python3-venv 
* python3-dev
* mysql-server 
* postfix 
* supervisor 
* nginx 
* git

``` bash
$ sudo apt-get -y update
$ sudo apt-get -y install python3 python3-venv python3-dev
$ sudo apt-get -y install mysql-server postfix supervisor nginx git
```
